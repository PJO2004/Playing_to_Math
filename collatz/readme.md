# Collatz-conjecture
## 간단설명

코라스 추측 : 1937년 처음으로 이 추측을 제기한 로타르 콜라츠의 이름을 딴것 , 3n+1추측, 올람 추측, 혹은 헤일스톤(우박)수열 등 여러이름으로 불림
- 짝수일시 2로 나눔
- 홀수라면 3을 곱하고 1을 더함
- 1이면 조작을 멈추고, 1이 아니면 첫 번째 단계로 돌아간다.
